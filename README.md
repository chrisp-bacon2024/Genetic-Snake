# Genetic Snake

Interactive Genetic Algorithm Snake AI platform. This phase includes a playable keyboard-controlled Snake game with a live control panel for visualizing direction input.

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Run

```bash
python src/main.py
```

## Controls

| Key | Action |
|-----|--------|
| Arrow keys | Move the snake |
| R | Restart after game over |
| Escape | Quit |

The left panel highlights arrow keys while they are pressed, mirroring how AI direction choices will be displayed later.
